This slide introduces the formal idea of composition of functions. It explains that composition involves applying one function to the result of another. In other words, the output of the inner function becomes the input for the outer function. It also reminds the reader that this only works if the inner function’s output is valid for the outer function. In this example, two functions are defined, one as a list of ordered pairs and the other as a table. The student is asked to perform function composition, using outputs from one to find the correct inputs for the next. It’s a great way to visualize how function composition works when not using equations. This slide shows the process of solving the composition problems from the previous example. This reinforces how the output of one function flows into the other, and helps learners see how tables and lists can work in composition.
